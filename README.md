# Task 5 - Exploratory Data Analysis (EDA)

## Objective

Perform exploratory data analysis to identify patterns, trends, and relationships in the Titanic dataset.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Analysis Performed

* Dataset overview using info() and describe()
* Missing value analysis
* Survival analysis
* Passenger class analysis
* Age distribution analysis
* Fare outlier detection
* Correlation heatmap
* Pairplot and scatterplot analysis

## Key Findings

* First-class passengers had the highest survival rates.
* Fare and passenger class influenced survival probability.
* Most passengers were aged between 20 and 40 years.
* Fare distribution contained significant outliers.
* Several features showed meaningful correlations.

## Outcome

Gained insights into passenger demographics, survival patterns, and feature relationships through statistical and visual exploration.
